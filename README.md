## Social Media Platform API
Backend social media system built using ASP.NET Core 6, Entity Framework Core, and SQL Server, with features for users, posts, comments, and secure authentication.

### Tech Stack:
C#, ASP.NET Core 6, Web API, EF Core, SQL Server, JWT, LINQ

### Main Features:
- User registration & login
- Create/Edit/Delete posts
- Comment system
- Secure authentication with JWT
- Clean database design
